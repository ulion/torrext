What?
No torrent files hosted here.
No trackers here either.
Data in the database is not indexed so searching it is impossible.
Torrents are identified only by info_hash.

API?
not a problem...
Send HTTP POST requests to /upload
if you want to send a file just put it in the "file" argument
if you prefer that torrext fetches the file from an http(s)
server put the url to the file in "url argument"

if everything goes has planed you'll get a 302 redirect to the page of the submited torrent
in case of errors you'll get a message

because torrext is still being developed it is best to check the code and see what messages are returned in each case: upload.py
